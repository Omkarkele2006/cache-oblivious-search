# Cache-Oblivious Search Structures

## Project Title
Design and Empirical Evaluation of Cache-Oblivious Search Structures for Modern Memory Hierarchies

## Objective
This project aims to implement and empirically evaluate cache-oblivious data structures such as van Emde Boas layouts and succinct trees against traditional cache-aware structures like Binary Search Trees and B-Trees.

## Tech Stack
- C++ (Core algorithms and benchmarking)
- Python (Data analysis and visualization)
- SQLite/PostgreSQL (Data storage)
- Bash (Automation)

## Research Goals
- Measure cache efficiency across memory hierarchies
- Compare runtime vs cache misses
- Evaluate performance under different workloads (uniform, Zipfian)
